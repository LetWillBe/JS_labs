Атрибут Async

скрипт начинает загружаться в фоновом режимке как только до него доходит страница

но запускается как только будет загружен

остальные скрипты будут загружаться не ожидая загрузки скриптов с async

в этом случае порядок выполнения скриптов может быть различным, т.к. первее будут запускаться те которые первее загрузятся
